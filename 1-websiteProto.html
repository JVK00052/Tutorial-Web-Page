Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure-a set of statements that performs a task of calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it

Objects

JavaScript is designed on a simple object-based paradigm. An object is a collection of properties, and a property is an association between a name (or key) and a value. 


        let randomObject = {

For Loops

There are many different kinds of loops, but they all essentially do the same thing, they repeat an action some number of times. A For Loop specifically repeats until a spcified condition evaluates to false. 


            for (var o = 20; o > 1; o--)
                console.log(o)
